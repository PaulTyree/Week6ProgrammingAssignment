# Week6ProgrammingAssignment

This user takes in the user's first name, asks the user how many days they went birding, how many hours they went birding, how many birds they saw that week, and what season it is.

Inputs in this program have failsafe loops to guarantee that there are no errors.

After collecting all true data, the program outputs a table of the data that is collected and sends it to a file called report.txt
